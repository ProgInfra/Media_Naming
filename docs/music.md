## Naming Rules

- Always **lowercase** (**NOT** Event Name **BUT** event name)
- Replace of **space** by **underline** (**NOT** event name **BUT** event_name)
- Multiple **zero digit** before number (**NOT** event_name_YYYY-MM-DD_1 **BUT** event_name_YYYY-MM-DD_001)
